Tutorial 4 Exercises and Solutions

4001CMD - Mathematical Skills for Computing Professionals

Functions

Problem 1

Compute values of recursive $s(n)$ and prove closed form.

View Solution
From recursion: $s(1)=1,s(2)=2,s(3)=3,s(4)=4,s(5)=5,s(6)=6$. Conjecture $s(n)=n$. Induct using $s(k+1)=2s(k)-s(k-1)=2k-(k-1)=k+1$.

Problem 2

Classify $p,q,r$ as injective/surjective/bijective.

View Solution

$p$ is neither injective nor surjective (repeat value 4; misses 1).

$q$ is injective and surjective, hence bijective.

$r$ is surjective but not injective (1 and 4 both map to a).

Problem 3

Classify $f(n)=n^3$, $g(n)=n+3$, and piecewise $h$.

View Solution
$f$ is injective but not surjective on $\mathbb N$. $g$ is injective but not surjective (1,2,3 have no preimage). $h$ swaps odd/even neighbors, so it is bijective.

Problem 4

Analyze Collatz-style recursive map.

View Solution
For $u(1)=1$, sequence reaches $1\to4\to2\to1$, so not injective and not surjective. For $u(1)=5$, it reaches 16,8,4,2,1 and same loop. Typical other starts also eventually reach this loop in tested examples, so still neither injection nor surjection.

Problem 5

Compare $t\circ s$ and $s\circ t$ for $s(x)=x+1,t(x)=2x$.

View Solution
$(s\circ t)(x)=2x+1$, while $(t\circ s)(x)=2x+2$, so they differ.

Problem 6

Show $(g\circ f)^{-1}=f^{-1}\circ g^{-1}$.

View Solution
Compute both sides: $(f^{-1}\circ g^{-1})\circ(g\circ f)=id$ and $(g\circ f)\circ(f^{-1}\circ g^{-1})=id$ by associativity and inverse identities.

Problem 7

Show $f$ has left inverse iff $f$ injective.

View Solution
If $l\circ f=id_X$, then $f(x_1)=f(x_2)\Rightarrow x_1=x_2$. Conversely, if $f$ is injective, define inverse on image $f(X)$ and extend arbitrarily to $Y$; then $l\circ f=id_X$.

Problem 8

For given $f$ on $\{1,2,3,4,5\}$, show idempotence and find $g$ with $f\circ g=f$ and $g\circ f=f$.

View Solution

(a) Direct check gives $f(f(x))=f(x)$ for all $x$.

(b) One valid choice: $g(1)=1,g(2)=2,g(3)=5,g(4)=4,g(5)=3$; also identity works. Then both compositions equal $f$.

Problem 9

Construct bijections on $\{1,2,3\}$ with orders 2 and 3.

View Solution
Take $s=(1\ 2)$ so $s\circ s=id$ and $s^{-1}=s$. Take $t=(1\ 2\ 3)$ so $t^3=id$, $t\ne id$, and $t^{-1}=t^2$.

Problem 10

Count bijections $S\to S$ and those with $f=f^{-1}$ for $S=\{1,2,3\}$.

View Solution
There are $3!=6$ bijections. Involutions are identity + three transpositions, so 4 satisfy $f=f^{-1}$.

Cardinality and Counting

Problem 11

Uniqueness of finite cardinality.

View Solution
$g\circ f^{-1}:\{1,..,m\}\to\{1,..,n\}$ is bijective, so injective; pigeonhole gives $m\le n$. Symmetrically $n\le m$. Hence $m=n$.

Problem 12

List all functions $\{1,2,3\}\to\{0,1\}$.

View Solution
Example: constant 0 function. Total count is $2^3=8$; each function corresponds to triple $(f(1),f(2),f(3))\in\{0,1\}^3$.

Problem 13

Count all functions $X\to Y$ with $|X|=m,|Y|=2$.

View Solution
Answer $2^m$. Induction: base $m=1$ has 2 functions. Step: add one element of $X$; each existing function extends in 2 ways.

Problem 14

Assign 4 students to 7 tutors without reuse.

View Solution
Number of injective assignments: $7\cdot6\cdot5\cdot4=\frac{7!}{3!}=840$.

Problem 15

Minimum sock draws to guarantee a matching pair.

View Solution
With 2 colors, worst case first two are different; third draw guarantees a pair. So 3.

Problem 16

12 naturals imply two differ by multiple of 11.

View Solution
Map each number to remainder mod 11 in $\{0,...,10\}$. 12 values into 11 classes => collision. Equal remainders mean difference divisible by 11.

Problem 17

Greatest odd-divisor map and multiples.

View Solution
$|Y|=n$. If $|X|\ge n+1$, $f:X\to Y$ is not injective. If $f(x_1)=f(x_2)=y$, write $x_i=2^{k_i}y$; then larger one is a power-of-2 multiple of the smaller.

Problem 18

Find $X$ of size $n$ in $\{1,...,2n\}$ with no divisibility relation.

View Solution
Choose $X=\{n+1,...,2n\}$. If $k,l\in X$ and $k=ml$ with $m\ge2$, then $k\ge2(n+1)>2n$, contradiction.

Problem 19

Show two sets are countably infinite by explicit bijections.

View Solution
(a) $n\mapsto n+10^6-1$ bijects $\mathbb N$ to $\{m\ge10^6\}$. (b) $n\mapsto3n$ bijects $\mathbb N$ to multiples of 3.

Problem 20

If $X\subseteq Y$ and $X$ infinite, prove $Y$ infinite.

View Solution
Contrapositive: if $Y$ finite, every subset $X\subseteq Y$ is finite. Therefore if $X$ infinite, $Y$ cannot be finite.

Problem 21

Non-empty subset of $\mathbb N$ with no greatest element and converse statements.

View Solution

(a) By well-ordering, choose $x_1$ as the least element of $X$. Since $X$ has no greatest element, $X\setminus\{x_1\}$ is non-empty; choose its least element $x_2$. Repeating gives a sequence with $x_1 < x_2 < \cdots$ and, in general, $x_{n+1} > x_n$.

(b) The sequence has infinitely many distinct terms in $X$, so $X$ is infinite.

(c) Conversely, if $X$ were infinite and had a greatest element $M$, then $X\subseteq\{1,\ldots,M\}$, which is finite, contradiction. So infinite subsets of $\mathbb N$ have no greatest member.
