Problem 1: Proof of the Addition Principle (PIE)

Question: Write out a proof (using induction on $n$) of the fact that if $A_{1},A_{2},\dots,A_{n}$ are disjoint finite sets, then $\#(A_{1} \cup A_{2} \cup \dots \cup A_{n}) = \#(A_{1}) + \#(A_{2}) + \dots + \#(A_{n})$.

Solution: Proof by Induction on $n$

Base Case ($n=2$):

For two disjoint sets, $A_1$ and $A_2$, the formula is $\#(A_1 \cup A_2) = \#(A_1) + \#(A_2)$. This is the definition of the basic Addition Principle for disjoint sets, so the base case holds.

Inductive Hypothesis:

Assume the formula holds for some integer $k \ge 2$. That is, for any $k$ disjoint finite sets $A_1, A_2, \dots, A_k$:

\[\#(A_{1} \cup A_{2} \cup \dots \cup A_{k}) = \#(A_{1}) + \#(A_{2}) + \dots + \#(A_{k})\]

Inductive Step ($n=k+1$):

Consider $k+1$ disjoint finite sets $A_1, A_2, \dots, A_{k+1}$. We group the first $k$ sets as a single set $B = A_{1} \cup A_{2} \cup \dots \cup A_{k}$.

The union of $k+1$ sets can be written as the union of two disjoint sets, $B$ and $A_{k+1}$:

$$\#(A_{1} \cup \dots \cup A_{k+1}) = \#(B \cup A_{k+1})$$

Applying the Base Case (or Basic Addition Principle) to the two disjoint sets $B$ and $A_{k+1}$:

$$\#(B \cup A_{k+1}) = \#(B) + \#(A_{k+1})$$

Now, substitute the Inductive Hypothesis for $\#(B)$:

$$\#(B) = \#(A_{1}) + \#(A_{2}) + \dots + \#(A_{k})$$

Therefore:

\[\#(A_{1} \cup \dots \cup A_{k+1}) = (\#(A_{1}) + \dots + \#(A_{k})) + \#(A_{k+1})\]

This is the required formula for $n=k+1$. By the Principle of Mathematical Induction, the formula holds for all finite $n \ge 2$.

Problem 2: Inclusion-Exclusion Principle (Languages)

Question: In a class of 67 mathematics students, 47 can read French (F), 35 can read German (G) and 23 can read both. (a) How many can read neither language? (b) If furthermore 20 can read Russian (R), of whom 12 can also read French, 11 read German also and 5 read all three, how many cannot read any of the three languages?

Solution: Applying PIE

Part (a): French and German

Total students $N = 67$. Given: $\#F = 47$, $\#G = 35$, $\#(F \cap G) = 23$.

Students reading at least one language ($\#(F \cup G)$) using PIE for two sets:

\[\#(F \cup G) = \#F + \#G - \#(F \cap G)\] \[\#(F \cup G) = 47 + 35 - 23 = 59\]

Students reading neither language:

$$\text{Neither} = N - \#(F \cup G) = 67 - 59 = 8$$

Part (b): French, German, and Russian

Given: $\#R = 20$, $\#(F \cap R) = 12$, $\#(G \cap R) = 11$, $\#(F \cap G \cap R) = 5$.

Students reading at least one language ($\#(F \cup G \cup R)$) using PIE for three sets:

\[\#(F \cup G \cup R) = (\#F + \#G + \#R) - (\#(F \cap G) + \#(F \cap R) + \#(G \cap R)) + \#(F \cap G \cap R)\] \[\#(F \cup G \cup R) = (47 + 35 + 20) - (23 + 12 + 11) + 5\] \[\#(F \cup G \cup R) = 102 - 46 + 5 = 61\]

Students reading none of the three languages:

$$\text{None} = N - \#(F \cup G \cup R) = 67 - 61 = 6$$

Answer: (a) 8 students, (b) 6 students.

Problem 3 (*): Permutations with Restrictions (PIE)

Question: Find the number of ways of arranging the letters A, E, M, O, U, Y in a sequence in such a way that neither of the words ME nor YOU occur. (Hint: Apply the sieve principle.)

Solution: PIE on Permutations

Total letters $n=6$. Total arrangements $\#(S) = 6! = 720$.

  • Let $P_{ME}$ be the set of arrangements containing the block ME.
  • Let $P_{YOU}$ be the set of arrangements containing the block YOU.

1. Arrangements containing ME ($\#P_{ME}$)

Treat ME as one unit. We arrange 5 units: $\mathbf{ME}, A, O, U, Y$.

$$\#P_{ME} = 5! = 120$$

2. Arrangements containing YOU ($\#P_{YOU}$)

Treat YOU as one unit. We arrange 4 units: $\mathbf{YOU}, A, E, M$.

$$\#P_{YOU} = 4! = 24$$

3. Arrangements containing ME and YOU ($\#(P_{ME} \cap P_{YOU})$)

We are arranging the 3 units: $\mathbf{ME}, \mathbf{YOU}$, and $A$.

$$\#(P_{ME} \cap P_{YOU}) = 3! = 6$$

4. Final Calculation

Arrangements with at least one bad word using PIE:

\[\#(P_{ME} \cup P_{YOU}) = \#P_{ME} + \#P_{YOU} - \#(P_{ME} \cap P_{YOU})\] \[\#(P_{ME} \cup P_{YOU}) = 120 + 24 - 6 = 138\]

The number of ways neither word occurs is the total minus the bad arrangements:

$$\text{Result} = 720 - 138 = 582$$

Answer: 582 ways.

Problem 4: Ordered Selection with Repetition (Flags)

Question: How many national flags can be constructed from three equal vertical strips, using the colours red, white, blue, and green? (Colours can be repeated, and order matters.)

Solution: Permutations with Replacement

This is a problem of ordered selection with repetition ($n^k$).

  • Number of choices (colors) $n=4$.
  • Number of selections (strips) $k=3$.

The total number of possible flags is:

$$\text{Total Flags} = 4^3 = 64$$

Problem 5: Power Set and Binary Correspondence

Question: Write down all of the subsets of the set $X=\{a,b,c\}$. Use the correspondence between $\mathcal{P}(X)$ and $\{0,1\}^{3}$ discussed in lecture to check that your list is complete.

Solution: Power Set and Binary Strings

The set $X$ has 3 elements, so the power set $\mathcal{P}(X)$ must have $2^3 = 8$ subsets. The binary string correspondence uses a 1 for inclusion and a 0 for exclusion for the elements $(a, b, c)$.

Subset Binary String (a, b, c)
$\emptyset$ (The empty set) 000
$\{a\}$ 100
$\{b\}$ 010
$\{c\}$ 001
$\{a, b\}$ 110
$\{a, c\}$ 101
$\{b, c\}$ 011
$\{a, b, c\}$ (The set itself) 111

The set of all binary strings of length 3, $\mathbf{\{0,1\}^{3}}$, contains $2^3=8$ unique strings, confirming the list of subsets is complete.

Problem 6: Ordered Selection without Repetition (Batting Order)

Question: In how many ways can we select a batting order of 11 from a pool of 14 cricketers?

Solution: Permutations

This is a permutation problem ($P(n, k)$) because the order of the players in the batting lineup matters, and repetition is not allowed.

  • Total players $n=14$.
  • Players selected $k=11$.

The number of ways is:

\[P(14, 11) = \frac{14!}{(14-11)!} = \frac{14!}{3!}\] \[P(14, 11) = 14 \times 13 \times 12 \times \dots \times 4 = 1,452,972,864,000\]

Answer: 1,452,972,864,000 ways.

Problem 8: The Four Selection Types

Question: Suppose $X=\{a,b,c\}$. How many selections of 2 elements is are possible from $X$, when those selections are: (a) ordered and repeating? (b) ordered and non-repeating? (c) unordered and repeating? (d) unordered and non-repeating?

Solution: Applying Counting Formulas

Set size $n=|X|=3$. Number of selections $k=2$.

(a) Ordered and Repeating

$$\text{Formula}: n^k$$ $$\text{Number} = 3^2 = 9$$

(b) Ordered and Non-Repeating (Permutations)

$$\text{Formula}: P(n, k)$$ $$\text{Number} = P(3, 2) = 3 \times 2 = 6$$

(c) Unordered and Repeating (Stars and Bars)

$$\text{Formula}: \binom{n+k-1}{k}$$ $$\text{Number} = \binom{3+2-1}{2} = \binom{4}{2} = 6$$

(d) Unordered and Non-Repeating (Combinations)

$$\text{Formula}: \binom{n}{k}$$ $$\text{Number} = \binom{3}{2} = 3$$

Answer: (a) 9, (b) 6, (c) 6, (d) 3.

Problem 9: Exponential Growth (Key Depths)

Question: If there are eight possible depths, how many positions are required to make one million different keys? (Hint: To facilitate the calculation, use the fact that $2^{10}$ is slightly greater than 1000.)

Solution: Using Powers of Two

Let $n=8$ be the number of possible depths, and $k$ be the number of positions. Total unique keys $= 8^k$. We require $8^k \ge 1,000,000$.

Since $8 = 2^3$, we can rewrite the equation as $(2^3)^k = 2^{3k}$. We need $2^{3k} \ge 10^6$.

Using the hint, $10^6 = (10^3)^2 \approx (2^{10})^2 = 2^{20}$.

We need the exponent $3k$ to be at least 20:

\[3k \ge 20 \implies k \ge \frac{20}{3} \approx 6.67\]

Since $k$ must be an integer, the smallest number of positions required is 7.

Answer: 7 positions are required ($8^7 = 2,097,152$).

Problem 10: Permutations of Symbols

Question: How many four-letter words can be made from an alphabet of 10 symbols if there are no restrictions on spelling except that no letter can be used more than once?

Solution: Permutations without Replacement

This is an ordered selection without repetition problem. The word length $k=4$ and total symbols $n=10$.

\[P(10, 4) = \frac{10!}{(10-4)!} = 10 \times 9 \times 8 \times 7 = 5040\]

Answer: 5040 four-letter words.

Problem 11: Unordered Selection with Repetition (Indistinguishable Dice)

Question: Show that when three indistinguishable dice are thrown, there are 56 possible outcomes. What is the number of outcomes when $n$ indistinguishable dice are thrown?

Solution: Stars and Bars

Part 1: Three Indistinguishable Dice

We are selecting $k=3$ results from $n=6$ possible faces, with repetition and order not mattering. We use $\binom{n+k-1}{k}$:

\[\text{Outcomes} = \binom{6+3-1}{3} = \binom{8}{3} = \frac{8 \times 7 \times 6}{3 \times 2 \times 1} = 56\]

Part 2: $n$ Indistinguishable Dice

The number of items to choose is $k=n$ (the dice), and the number of types is $n=6$ (the faces).

\[\text{Outcomes} = \binom{6+n-1}{n} = \binom{n+5}{n} \quad \text{or equivalently, } \binom{n+5}{5}\]

Answer: 56 outcomes for 3 dice; $\binom{n+5}{5}$ for $n$ dice.

Problem 12 (*): Stars and Bars Proof

Question: Show that the number of $n$-tuples $(x_{1},\dots,x_{n})$ of non-negative integers which satisfy the equation $x_{1}+x_{2}+\dots+x_{n}=r$ is $\binom{n+r-1}{r}$.

Solution: Combinatorial Proof (Stars and Bars)

This equation represents the number of ways to distribute $r$ identical objects (stars, $\star$) into $n$ distinct bins ($x_1$ through $x_n$). To separate the $n$ bins, we need $n-1$ indistinguishable separators (bars, $|$).

The total number of objects we are arranging is the sum of the stars and the bars:

$$\text{Total Objects} = r + (n-1) = n+r-1$$

Any arrangement of these $n+r-1$ objects corresponds to a unique solution. The problem reduces to finding the number of ways to choose the $r$ positions for the stars (or $n-1$ positions for the bars) from the total $n+r-1$ positions.

\[\text{Number of solutions} = \binom{n+r-1}{r}\]

This proves the formula for the number of non-negative integer solutions.

Problem 13: Binomial Theorem Formulas

Question: Write out the formulas for $(1+x)^{8}$ and $(1-x)^{8}$.

Solution: Binomial Expansion

The general Binomial Theorem states: $$(a+b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k$$

A. Formula for $(1+x)^{8}$

With $a=1$ and $b=x$, the expansion is:

\[(1+x)^8 = \sum_{k=0}^{8} \binom{8}{k} x^k\] \[(1+x)^8 = 1 + 8x + 28x^2 + 56x^3 + 70x^4 + 56x^5 + 28x^6 + 8x^7 + x^8\]

B. Formula for $(1-x)^{8}$

With $a=1$ and $b=-x$, the signs of the odd powers of $x$ alternate:

\[(1-x)^8 = \sum_{k=0}^{8} \binom{8}{k} (-1)^k x^k\] \[(1-x)^8 = 1 - 8x + 28x^2 - 56x^3 + 70x^4 - 56x^5 + 28x^6 - 8x^7 + x^8\]

Problem 14: Finding Specific Coefficients

Question: Calculate the coefficients of: (a) $x^{5}$ in $(1+x)^{11}$ (b) $a^{2}b^{8}$ in $(a+b)^{10}$ (c) $a^{6}b^{6}$ in $(a^{2}+b^{3})^{5}$ (d) $x^{3}$ in $(3+4x)^{6}$

Solution: Coefficient Calculation

(a) Coefficient of $x^{5}$ in $(1+x)^{11}$

Using $\binom{n}{k}$, where $n=11$ and $k=5$:

$$\text{Coefficient} = \binom{11}{5} = \frac{11 \times 10 \times 9 \times 8 \times 7}{5 \times 4 \times 3 \times 2 \times 1} = 462$$

(b) Coefficient of $a^{2}b^{8}$ in $(a+b)^{10}$

Using $\binom{n}{k}$, where $n=10$ and $k=8$ (for $b^8$):

$$\text{Coefficient} = \binom{10}{8} = \binom{10}{2} = 45$$

(c) Coefficient of $a^{6}b^{6}$ in $(a^{2}+b^{3})^{5}$

General term: $\binom{5}{k} (a^2)^{5-k} (b^3)^k = \binom{5}{k} a^{10-2k} b^{3k}$.

Set $3k=6 \implies k=2$. The coefficient is $\binom{5}{2}$.

$$\text{Coefficient} = \binom{5}{2} = 10$$

(d) Coefficient of $x^{3}$ in $(3+4x)^{6}$

Set $k=3$. The term is $\binom{6}{3} (3)^{6-3} (4x)^3 = \binom{6}{3} 3^3 4^3 x^3$.

\[\text{Coefficient} = \binom{6}{3} \cdot 3^3 \cdot 4^3 = 20 \cdot 27 \cdot 64 = 34,560\]

Answer: (a) 462, (b) 45, (c) 10, (d) 34,560.
