Basic Combinatorics

Problem 1

Write out a proof (using induction on $n$) of the fact that if $A_1,A_2,\dots,A_n$ are disjoint finite sets, then $\#(A_1 \cup A_2 \cup \dots \cup A_n)=\#(A_1)+\#(A_2)+\dots+\#(A_n)$.

View Solution

Proof by induction on $n$.

Base case ($n=2$): For disjoint $A_1,A_2$, $\#(A_1\cup A_2)=\#(A_1)+\#(A_2)$ by the addition principle.

Induction hypothesis: Assume true for $n=k$ disjoint sets.

Induction step: For $k+1$ disjoint sets, let $B=A_1\cup\cdots\cup A_k$. Then $B$ and $A_{k+1}$ are disjoint, so

$$\#(B\cup A_{k+1})=\#(B)+\#(A_{k+1}).$$

By the hypothesis, $\#(B)=\#(A_1)+\cdots+\#(A_k)$, hence

$$\#(A_1\cup\cdots\cup A_{k+1})=\#(A_1)+\cdots+\#(A_k)+\#(A_{k+1}).$$

Therefore true for all $n\ge 2$.

Problem 2

In a class of 67 mathematics students, 47 can read French, 35 can read German and 23 can read both. (a) How many can read neither language? (b) If 20 can read Russian, 12 can also read French, 11 also read German, and 5 read all three, how many cannot read any of the three languages?

View Solution

(a) Using PIE for two sets:

$$|F\cup G|=|F|+|G|-|F\cap G|=47+35-23=59.$$

Neither language: $67-59=8$.

(b) Using PIE for three sets:

$$|F\cup G\cup R|=|F|+|G|+|R|-|F\cap G|-|F\cap R|-|G\cap R|+|F\cap G\cap R|$$ $$=47+35+20-23-12-11+5=61.$$

Cannot read any of the three: $67-61=6$.

Problem 3

Find the number of ways of arranging the letters A, E, M, O, U, Y in a sequence so that neither the word ME nor YOU occurs. (Hint: apply the sieve principle / inclusion-exclusion.)

View Solution

Total permutations: $6!=720$.

Let $A$ be arrangements containing ME, and $B$ containing YOU.

$|A|=5!=120$ (treat ME as one block).

$|B|=4!=24$ (treat YOU as one block).

$|A\cap B|=3!=6$ (blocks ME, YOU, and A).

Bad arrangements: $|A\cup B|=120+24-6=138$.

Required: $720-138=582$.

Problem 4

How many national flags can be constructed from three equal vertical strips, using the colours red, white, blue, and green? (Colours can be repeated and order matters.)

View Solution

Each of 3 strip positions has 4 choices, independently.

$$4^3=64.$$

So there are 64 possible flags.

Problem 5

Write down all subsets of $X=\{a,b,c\}$. Use the correspondence between $\mathcal{P}(X)$ and $\{0,1\}^3$ to check your list is complete.

View Solution

Subsets are:

$\emptyset,\{a\},\{b\},\{c\},\{a,b\},\{a,c\},\{b,c\},\{a,b,c\}$.

There are 8 subsets, matching $2^3=8$ and the 8 binary strings in $\{0,1\}^3$.

Problem 6

In how many ways can we select a batting order of 11 from a pool of 14 cricketers?

View Solution

Order matters and no repetition, so use permutations:

$$P(14,11)=\frac{14!}{(14-11)!}=\frac{14!}{3!}=1,452,972,864,000.$$

Problem 8

Suppose $X=\{a,b,c\}$. How many selections of 2 elements are possible when selections are: (a) ordered and repeating, (b) ordered and non-repeating, (c) unordered and repeating, (d) unordered and non-repeating?

View Solution

$n=3, k=2$.

(a) Ordered, repeating: $n^k=3^2=9$.

(b) Ordered, no repetition: $P(3,2)=3\cdot2=6$.

(c) Unordered, repeating: $\binom{n+k-1}{k}=\binom{4}{2}=6$.

(d) Unordered, no repetition: $\binom{3}{2}=3$.

Problem 9

If there are eight possible depths, how many positions are required to make one million different keys? (Hint: $2^{10}$ is slightly greater than 1000.)

View Solution

Need $8^k\ge 10^6$. Since $8=2^3$, this is $2^{3k}\ge 10^6$.

Using $2^{10}\approx 10^3$, we get $10^6\approx 2^{20}$, so $3k\ge 20$.

Thus $k\ge 6.67$, so smallest integer is $k=7$.

Problem 10

How many four-letter words can be made from an alphabet of 10 symbols if no letter can be used more than once?

View Solution

Permutation without repetition:

$$P(10,4)=10\cdot 9\cdot 8\cdot 7=5040.$$

Problem 11

Show that when three indistinguishable dice are thrown, there are 56 possible outcomes. What is the number of outcomes when $n$ indistinguishable dice are thrown?

View Solution

Indistinguishable dice means unordered outcomes with repetition allowed from 6 faces.

For 3 dice:

$$\binom{6+3-1}{3}=\binom{8}{3}=56.$$

For $n$ dice:

$$\binom{6+n-1}{n}=\binom{n+5}{n}=\binom{n+5}{5}.$$

Problem 12

Show that the number of $n$-tuples $(x_1,\dots,x_n)$ of non-negative integers satisfying $x_1+x_2+\dots+x_n=r$ is $\binom{n+r-1}{r}$.

View Solution

Stars and bars: represent the $r$ units as stars and separate $n$ variables with $n-1$ bars.

Total symbols: $r+n-1$.

Choose positions of the $r$ stars (or $n-1$ bars):

$$\#=\binom{n+r-1}{r}=\binom{n+r-1}{n-1}.$$

Problem 14

Calculate the coefficients of: (a) $x^5$ in $(1+x)^{11}$, (b) $a^2b^8$ in $(a+b)^{10}$, (c) $a^6b^6$ in $(a^2+b^3)^5$, (d) $x^3$ in $(3+4x)^6$.

View Solution

(a) Coefficient of $x^5$ in $(1+x)^{11}$ is $\binom{11}{5}=462$.

(b) Coefficient of $a^2b^8$ in $(a+b)^{10}$ is $\binom{10}{8}=\binom{10}{2}=45$.

(c) In $(a^2+b^3)^5$, general term is $\binom{5}{k}a^{10-2k}b^{3k}$. Set $3k=6\Rightarrow k=2$, coefficient $\binom{5}{2}=10$.

(d) In $(3+4x)^6$, coefficient of $x^3$ is:

$$\binom{6}{3}3^{3}4^{3}=20\cdot 27\cdot 64=34,560.$$
