\section{Propositions and Proofs}
\subsection{Terminology}
			
		\begin{frame}{Statements and proofs: definitions}
			\begin{itemize}\itemsep5mm
				\item A \textbf{proposition} is a statement that is either true or false.\p
				\item An \textbf{axiom}, \textbf{postulate}, or \textbf{assumption} is a statement that is taken to be true, to serve as a premise or starting point for further reasoning and arguments. \p
				\item In classic philosophy, an axiom is a statement that is so evident or well-established, that it is \textit{accepted without controversy or question}.\p
				\item A true statement is called a \textbf{theorem}. \p
				\item A false statement is a \textbf{nontheorem}. Its negation is a theorem. \p
				\item A mathematically and logically valid explanation for why a theorem is true is a \textbf{proof}. 
			\end{itemize}
		
		
%		But what is \textbf{proof} anyway? \p 
		
	\end{frame}

\begin{frame}{Compound propositions: IFF "\textbf{if and only if... then}" ($\iff$) }
	\p
	
\begin{itemize}\itemsep2mm
	\item The implication "\textbf{If} I am a vegetarian, \textbf{then} I won't have chicken tonight for dinner " certainly "restricts" vegetarians having chicken for a dinner.\p
	
	\item However non-vegetarians (last two lines in the truth-table) can have either chicken or anything else for dinner. The statement will still be True.\p
	
	\item What if we wanted to restrict the option of "\textit{not having a chicken}" for dinner to only vegetatians (i.e. \textit{non-vegetarians must eat chicken regardless})? \p
	
	\item In this case we must say "\textbf{Only if} I am a vegetarian, then I will not have chicken ..." \p
	
	\item Mathematicians make it more dramatic and instead of saying "\textbf{only if}" they say "\textbf{if and only if}" \p
	
	\item Notice that now if you are \textbf{not having chicken}, that means \textbf{you are certainly a vegetarian}. Hence both parts of this proposition (if part and then part) becomes equivalent! \p
	
	\item it becomes an implication both ways: "if you are vegetarian $\implies$ you didn't have chicken" and "if you didn't have chicken $\implies$ you are (certainly) a vegetarian". 
\end{itemize}

 



 
	
	
\end{frame}	
	
	
	
\begin{frame}{Compound propositions: IFF "\textbf{if and only if... then}" ($\iff$) }
	
	\vspace{0mm}
\begin{columns}[t]
		
	\begin{column}[t]{0.6\textwidth}
			\p
	The proposition \textbf{“$P$ if and only if $Q$”} asserts that \textbf{$P$ and $Q$ are logically equivalent}; that is, either both are true or both are false. \p In other words: $(P\implies Q) \land (Q \implies P)$ is equivalent to $P\iff Q$.
	\p
	\end{column}
		
	\begin{column}[t]{0.4\textwidth}
	\vspace{-3mm}
	\begin{examplen}
	$x^2-4 \ge 0 \iff |x| \ge 2 $
	\end{examplen}
	
	\p
	
	
	
	\end{column}		
	\end{columns}
	\vspace{2mm}
	
For some values of $x$, both inequalities are true. For other values of $x$, neither inequality is true. In every case, however, the proposition as a whole is true.\vspace{0mm}

\subsection{Existential statements}		
		\begin{frame}{Existential statements}
			\begin{itemize}
				\item Let's think back to our first example... 
			\end{itemize}
			\vspace{2mm}
			
			\textbf{Theorem.} 15 is a multiple of 3. \p
			\begin{itemize}
				\item To say that a (whole) number $x$ is a multiple of 3 means that there is some other (whole) number $n$ for which $3\times n = x$. \p
				\item So the theorem can be rewritten as: 
			\end{itemize}
			\vspace{2mm}
			
			
			\textbf{Theorem.} There is a whole number $n$ for which $3 \times n = 15$. \p
			\begin{itemize}
				\item This is an \emph{existential statement:} it is a statement that asserts the \textit{existence} of a number with a particular property (in this case, the property of the number is that its product with 3 is 15). 
			\end{itemize}
		\end{frame}


		
		\begin{frame}{Examples of existential statements}
			\p
			\begin{examplen}
				There exists ($\exists$) natural number $n$ such that $n^2 = 169$. ~~~ $\exists n\in \N : n^2=169$
			\end{examplen}
			\p
			\begin{itemize}
				\item Is this statement true? \p
				\begin{itemize}
					\item \emph{Yes!} This works for $n=13$, since $13^2 = 169$. \p
				\end{itemize}\p
			\end{itemize}
			\begin{examplen}
				There are numbers $a, b, c$ for which $a^2 + b^2 = c^2$. 
			\end{examplen}\p
			\begin{itemize}
				\item Is this statement true? \p
				\begin{itemize}
					\item \emph{Yes!} We can take $a=3$, $b=4$, $c=5$, since $3^2 + 4^2 = 9 + 16 = 25 = 5^2$, so $3^2 + 4^2 = 5^2$. \p
					\item This also works with $a=5$, $b=12$, and $c=13$: $5^2 + 12^2 = 25 + 144 = 169 = 13$. \p
				\end{itemize}
				\item Existential statements do not necessarily say that the object whose existence is asserted is \emph{unique}: there may be more than one in question (such as was the case in the last Example 11). 
			\end{itemize}
		\end{frame}
		
		\begin{frame}{Proving existential statements}\p
			\vspace{-2mm}
			\begin{itemize}
				\item \textbf{Question.} How do we prove that an existential statement like ``There are numbers $a,b,c$ for which $a^2 + b^2 = c^2$ is true? \p
				\item \textbf{Answer.} Find (at least one) example! In the above, $a=3$, $b=4$, $c=5$ does the trick, and shows that such an $a$, $b$, and $c$ do exist. \p
				\item In general, to prove an existence statement, only one example needs to be produced. \p
				\item But in some cases, proving the existence may be very difficult... \p
			\end{itemize}
			\begin{example}
				There exist numbers $a$, $b$, $c$ for which $a^3 + b^3 = c^3$. 
			\end{example}\p
			\begin{itemize}
				\item Is this statement true?\p ~ \textbf{NO!} \p... Actually, it took mathematicians hundreds of years to answer... \p  
				\item Another example: \\ \finger \textbf{Euler conjecture}:\fbox{$a^4+b^4+c^4=d^4$ has no solution for positive integers $a,b,c,d$}. \\ \vspace{1mm} It took 218 years to find a solution satisfying this equation and prove the proposition is false! ($a=95800,~b=217519,~c=414560,~d=422481$)
			\end{itemize}
		\end{frame}
		
		
		
		
		
\begin{frame}{Disproving existential statements}
		
	The true statement, it turns out, is: \p
		
		
		\begin{theorem}
			There are no numbers $a$, $b$, $c$ for which $a^3 + b^3 = c^3$.			
		\end{theorem}\p
		

			\begin{itemize}
				\item But how do you prove it? You certainly can't check it with every possible triple of numbers $a,b,c$. \p
				\item You need to show that something is true for \emph{every} choice of $a,b,c$: that $a^3 + b^3 \neq c^3$. \p
				\item So another way to write the statement is: 
			\end{itemize}
		
		\begin{theorem}
			For all numbers $a,b,c$: $a^3 + b^3 \neq c^3$.			
		\end{theorem}
		\p
		
	
			\begin{itemize}
				\item Statements that make claims about \textit{all} numbers are \textit{\textbf{universal statements}}. \p
				\item See the difference: An \textbf{existential statement} claims that a certain property holds for \textbf{some} number; perhaps one, perhaps several, perhaps all, but at least one. A \textbf{universal statement} claims that a certain property holds for \textbf{all} numbers. 
			\end{itemize}
		\end{frame}

\subsection{Universal Statements}
		
		\begin{frame}{Universal statements: examples}
			\begin{itemize}\itemsep5mm
				\item The statement ``For all numbers $a,b,c$: $a^3 + b^3 \neq c^3$ is exceptionally hard to prove. \p
				\item Actually, it's a specific case of a more general, very famous universal statement: \p
			\end{itemize}\vspace{2mm}
			
			\textbf{Theorem.} For all numbers $n$, there are no numbers $a,b,c$ for which $a^n + b^n = c^n$. 
			\vspace{3mm}
			
			\begin{itemize}\p
				\item Let's look at a simpler example: 
			\end{itemize}
			\begin{example}
				For all whole numbers $n$, the number $n^2 + n$ is even. 
			\end{example}\p
			\begin{itemize}
				\item Is this true? \pause 
				\item It turns out the answer is \emph{yes:}
			\end{itemize}
		\end{frame}
		
		\begin{frame}{Universal statements: examples}
%			\begin{itemize}
%				
%			\end{itemize}
			\textbf{Theorem.} For all numbers $n$ (i.e. $\forall n \in N$), the number $n^2 + n$ is even. \newline \p
			\textbf{Proof.} \p
			\begin{itemize}
				\item We need to show that there is an integer $k$ for which $n^2 + n = 2k$. \p
				\item A number $n$ is either even or odd. \p
				\item \emph{Case 1:} $n$ is even, say $n=2r$  \p
				\vspace{2mm}
				
				\begin{itemize}
					\item Then $n^2 + n = 4r^2 + 2r = 2(2r^2 + r)$, which is even: $\therefore ~ k = 2r^2 + r$.  \p
				\end{itemize}
				\item \emph{Case 2:} $n$ is odd, say $n=2r+1$.  \p
				\vspace{2mm}
				
				\begin{itemize}
					\item Then $n^2 = (2r+1)^2 = 4r^2 + 4r + 1$. \p So, $$n^2 + n = \p \underbrace{4r^2 + 4r + 1}_{n^2} + \underbrace{2r + 1}_{n} = \p 4r^2 + 6r + 2 = \p 2(2r^2 + 3r + 1).$$ \p So again, $n^2 + n$ is even: $k = 2r^2 + 3r + 1$. \qed 
				\end{itemize}
				\item This is often a common way of proving universal statements: show that there are \textbf{finitely many} cases, and show that it works in each such case. 
			\end{itemize}
		\end{frame}

\subsection{Proof by Contradiction}

\begin{frame}{Proof by Contradiction: The Method}
	Proof by contradiction, also known as \textit{reductio ad absurdum}, is a powerful indirect proof technique used to establish the truth of a statement.\vspace{3mm}
	\begin{itemize}
		\item \textbf{The Steps:}
		\begin{enumerate}\itemsep3mm
			\item Assume the \textbf{negation} of the statement you want to prove is true.
			\item Use a logical chain of reasoning to follow from this assumption.
			\item Arrive at a \textbf{contradiction}. This is a statement that is logically impossible, such as $x > 0$ and $x \leq 0$.
			\item Since a false premise must have led to this contradiction, our initial assumption must be false.
			\item Therefore, the original statement we wanted to prove must be true.
		\end{enumerate}
	\end{itemize}
\end{frame}

% The example slide
\begin{frame}{Proof by Contradiction: An Example}
	\begin{itemize}\itemsep3mm
		\item \textbf{Example: Proving $\sqrt{2}$ is Irrational}
		\vspace{3mm}
		\begin{enumerate}\itemsep3mm
			\item \textbf{Statement:} $\sqrt{2}$ is irrational.\p
			\item \textbf{Assume the opposite:} $\sqrt{2}$ is rational, \p i.e. $\sqrt{2}$ can be expressed as a rational fraction. \p 
			\item Then by definition, $\sqrt{2} = \frac{p}{q}$ where $p$ and $q$ are integers, $q \neq 0$, and $\frac{p}{q}$ is in its simplest form.\p
			\item Squaring both sides: $2 = \frac{p^2}{q^2} \implies 2q^2 = p^2$. This means $p^2$ is even, so $p$ must be even. \p
			\item Let $p = 2k$ for some integer $k$. Substituting this gives $2q^2 = (2k)^2 = 4k^2 \implies q^2 = 2k^2$. \p
			\item This means $q^2$ is even, so $q$ must also be even (any product with even number is even). \p
			\item \textbf{Contradiction!} We have shown that both $p$ and $q$ are even, which contradicts our initial assumption that the fraction $\frac{p}{q}$ was in its simplest form. \p
			\item Therefore, the original statement is true: $\sqrt{2}$ is irrational. 
		\end{enumerate}
	\end{itemize}
\end{frame}



\subsection{Summary on Proofs}		
		\begin{frame}{Summary: Techniques of proof}
			\begin{itemize}\itemsep4mm
				\item Let's review some of the basic proof techniques we've seen so far: \p
				\item \textbf{To prove an existential statement is true, produce an example}. 
				\begin{itemize}
					\item E.g., ``15 is a multiple of 3'' (one example is enough).
				\end{itemize}\p
				\item \textbf{To prove a universal statement is \underline{false}, produce a counterexample}. 
				\begin{itemize}
					\item E.g., ``For all numbers $n$, $4n-1$ is prime'' was proven false because it fails to be true when $n=4$. 
				\end{itemize}\p
				\item These can still be difficult to do in general. \p
				\item Another powerful technique is \emph{``\textbf{proof by contradiction}''}: assume the statement you're trying to prove is false, and arrive at a false conclusion. 
				\begin{itemize}
					\item E.g., ``20 is not a multiple of 3'' was proven by contradiction. \p 
				\end{itemize}
				\item This list is not exhaustive! There are many other types of proofs. Some are easier or harder than others. But there is no strategy that works for proving every true statement. 
			\end{itemize}
		\end{frame}
